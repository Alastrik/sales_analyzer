Универсальный анализатор продаж и цен

Простое, но мощное десктопное приложение на Python для анализа данных о продажах и ценах.  
Поддерживает три формата файлов (`.csv`, `.txt`, `.xlsx`), автоматически определяет структуру данных и строит реалистичные прогнозы.

Возможности

Автоматическое распознавание режима
- Режим продаж если файл содержит колонки `Date` и `Total`
  - Прогноз продаж на следующий месяц (среднее за последние 3 месяца)
  - Анализ корзины покупок (какие товары покупают вместе)
  - Графики динамика продаж, распределение чеков
- Режим цен если файл содержит `Date`, `Product` и `Price`
  - Прогноз цены на товар в следующем месяце
  - График динамики цены

Поддержка форматов
- `.csv` (с любой кодировкой UTF-8, Windows-1251)
- `.txt` (с разделителями `,`, `t`, `;`)
- `.xlsx` (первый лист Excel)

Визуализация
- Продажи по годам и месяцам
- Распределение сумм чеков (гистограмма)
- Динамика цен на товар
- Автоматический подбор ширины колонок в Excel-отчёте

Экспорт отчёта
Создаётся файл вида `ваш_файл_sales_report.xlsx` с листами
- `Данные` — исходные данные (даты в читаемом формате, без `######`)
- `Прогноз` — прогноз продаж или цены
- `Корзина` — совместные покупки (если применимо)

Требования

- Python 3.9+
- Windows  macOS  Linux
